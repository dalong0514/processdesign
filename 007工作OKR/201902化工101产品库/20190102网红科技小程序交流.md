## 收获

1、四个主页：2 个产品内容展示，1 个问题反馈，1 个个人中心。

2、原型定了，开发周期一个星期，给领导报两个星期。

3、后端主要用 PHP，Java 的成本要高很多。目前的版本 PHP 就可以了，目前的代码是比较精简的，不会出现什么很大的 bug。用 Java 是为了以后性能的优化，还有迭代的稳定性。小程序这一块目前还是 PHP 为主。

5、主要是做 C 端的，做分销、营销功能做的很多。
